The game would be played according to the users input
After the game is played, the modal of success or try again appears
In the modal of success, when the user swipes from left to right, we want to go to the next stage
In the modal of try again, when the user swipe from right to left, the user can come back to the game page to understand what they did wrong

Note that the swipe left and right navigation are buitl on game.ts
